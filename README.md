# source-view
(Basic) Functionality to view a python library as a graph 

Usage

```bash
$ python source_view.py ~/Documents/github/euclid/`
```
This would generate a file called as `euclid.html` at `~/Documents/github/`  
Visit https://nbviewer.org/github/sizhky/source-view/blob/main/euclid.html to see it interactive
